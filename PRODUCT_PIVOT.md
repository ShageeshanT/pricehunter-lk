# Product Pivot: Universal Price Range Finder

PriceHunter LK is not only for Young Protege or event budgets.

The core product is now a simple public-facing price range finder:

1. User enters an item name.
2. Agent searches configured websites and ecommerce-style sources.
3. App returns only the cheapest found price and the most expensive found price.
4. Each result includes the site/vendor name and source URL.

## MVP output

For each item:

- Cheapest site name
- Cheapest price
- Cheapest source URL
- Most expensive site name
- Most expensive price
- Most expensive source URL

No Excel export is required for the main product flow.
